# Korean_ABSA

Aspect-Based Sentiment Analysis  
Language: Korean  
Dataset: https://corpus.korean.go.kr/task/taskDownload.do?taskId=8&clCd=ING_TASK&subMenuId=sub02  
Baseline: https://github.com/teddysum/korean_ABSA_baseline

<br>

## How to use
### Train
`$ bash train.sh`

### Test
`$ bash test.sh`
